Для вывода строки на экран, используется системный вызов write.

Для обращения к системным вызовам, необходимо в регистр rax записать номер нужного системного вызова, аргументы передавать через регистры rdi, rsi, rdx, r10d, r8d, r9d. После заполнения регистров, необходимо использовать команду syscall. Системные вызовы записывают возвращаемое значение в rax.
Согласно ABI Linux x64, системный вызов write имеет номер 1.

Аргументы для write:
rdi: unsigned int fd
rsi: const char *buf
rdx: size_t count

syscall - обращение к системному вызову с номером, который записан в rax. (system call)
