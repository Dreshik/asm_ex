ABI linux для 32 битной и 64 битной архитектур отличается.

Используемые регистры:
	32 bit: eax, edi, ecx
	64 bit: rax, rdi, rcx
На 64 bit можно использовать eax, т.к он является первыми 32 битами rax. Аналогично с другими регистрами.

Номера системных вызовов тоже отличаются.
Номер системного вызова getpid
	32 bit: 0x14
	64 bit: 0x27

Обращение к системному вызову:
	32 bit: int 0x80
	64 bit: syscall
